# nv_v2
csgo sound esp
----
## usage
* clone this repository
* build x86 release
* set your ingame volume low, recommended under 0.3.
* copy "nvStereoApiI.dll" -> "C:\Program Files (x86)\NVIDIA Corporation\3D Vision\" folder
* start the game and play

----
## features
* no hooks
* no ingame functions called
